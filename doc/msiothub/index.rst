Microsoft IoTHub API
--------------------

This describes the various means of interfacing Microsoft Azure IoTHub to the Jitsuin Archivist
system.

There are 2 methods of ingressing Azure IoTHub devices into the Jitsuin Archivist system:

    1. Read current Microsoft IoThub devices into Archivist
    2. Subscription service that listens on an Microsoft Azure Service Bus Queue for IoTHub events.
