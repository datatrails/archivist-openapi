Administrator configuration
---------------------------

The Jitsuin Archivist is fully integrated with the `Microsoft Identity
Platform`_ (v2.0) using OAuth 2.0 and Open ID Connect. A small amount of Azure
administrative configuration is required to control user assignment and enable
non interactive API access. The following section details the necessary
configuration.

.. _`Microsoft Identity Platform`: https://docs.microsoft.com/bs-latn-ba/azure/active-directory/develop/v2-overview
